# send-it

Open-source sports performance tools for data-driven athletes.

Single-file browser apps — no install, no server, no account. Drop in your data and go.

## Tools

| App | Description | Live |
|-----|-------------|------|
| [PaceMaxing](./pacemaxing/) | Rowing interval analyzer — FIT file → pace, DPS, HR by quarter and sprint segment | [Open](https://goeppertvb.github.io/send-it/pacemaxing/) |

## Philosophy

- **Local-first.** All processing happens in your browser. Your training data never leaves your machine.
- **One file.** Each tool is a single self-contained HTML file. Download it, open it, use it offline.
- **Athlete-built.** These tools came out of real training analysis on the Schuylkill River. They solve problems we actually had.
